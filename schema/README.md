# Data Schemas & Interfaces

This directory is intended to house the core data definitions for `ajdigital.app`. 

As you design the backend, place your data models here:
- Prisma schemas (`schema.prisma` if opting for Prisma)
- Exported TypeScript interfaces and types for the frontend to consume.

By keeping schemas in a dedicated directory, AI agents can quickly understand the data relationships driving the application without sifting through business logic.
