# Site Architecture

## Purpose
Details the URL structure, sitemap hierarchy, and routing strategy for the entire web platform.

## Main Navigation
- Home (`/`)
- Services (`/services/*`)
- About (`/about`)
- Blog (`/blog/*`)

## Footer Links
- Service clusters
- Legal (Privacy Policy, Terms of Service)
- Social integrations

## Dynamic Routes
- Blog posts: `/blog/[slug]`
- Local SEO Pages: `/locations/[city]`
