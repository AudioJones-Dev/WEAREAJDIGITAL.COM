# Build Order

## Purpose
Provides a step-by-step sequential development checklist to maintain efficiency and handle dependencies properly.

## Phase 1: Infrastructure
- Repository setup.
- Global styles and theme configuration.
- Base layout components (Navbar, Footer).

## Phase 2: Core Routing
- Homepage framework.
- Core static pages (About, Contact).

## Phase 3: Content Engines
- Service pages implementation.
- CMS setup for Blog and Local SEO.

## Phase 4: Integrations & QA
- Forms and CRM/n8n hooks.
- SEO audits and schema validation.
- Final launch checks.
