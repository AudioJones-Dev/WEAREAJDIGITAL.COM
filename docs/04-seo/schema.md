# Schema Markup Types

## Purpose
Defines the Structured Data (JSON-LD) injected into specific pages to enhance search visibility.

## Organization Schema
- Applied to Homepage and About page.
- Includes Name, Logo, URL, Social Profiles, Contact Point.

## Service Schema
- Applied to all `/services/*` pages.
- Defines the service type, provider, and area served.

## FAQ Schema
- Used on Service Pages and Key Blog Posts.
- Maps direct Questions and Answers for rich snippets.

## Article Schema
- Applied via the Blog layout dynamically.
- Includes Headline, DatePublished, DateModified, Author, Publisher.
