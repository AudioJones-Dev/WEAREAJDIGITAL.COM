# Internal Linking Rules

## Purpose
Establishes a systematic approach to distributing page rank and guiding website visitors through our conversion funnels.

## Pillar → Cluster Linking
- Pillar pages (Service pages) must link down to cluster topics (Blog posts).
- Cluster topics must link back up to their parent Pillar.

## Anchor Text Rules
- Avoid generic anchors ("click here", "read more").
- Use descriptive, exact or partial match keywords.
- Vary anchor text to present a natural linking profile.

## CTA Linking
- In-content CTAs should accurately reflect the intent of the page.
- Always provide an "escape hatch" to the `/apply` page.
