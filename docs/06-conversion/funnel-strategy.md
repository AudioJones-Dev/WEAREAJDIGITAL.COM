# Funnel Strategy

## Purpose
Defines the overarching logic for moving a cold visitor to a qualified, booked lead.

## Traffic Sources
- Organic Search (SEO)
- Direct link clicks (Social, PR)
- Referral

## The Core Funnel
1. **Awareness:** Land on Homepage, Blog, or Local Page.
2. **Consideration:** Reads content, views Service Pages, checks Social Proof.
3. **Intent:** Clicks CTA to `/apply`.
4. **Qualification:** Fills out multi-step form.
5. **Action:** Books call or is placed in automated nurture sequence.

## Retention & Nurture
- Handling users who drop off before booking.
- Retargeting pixel implementation.
