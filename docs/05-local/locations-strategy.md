# Local SEO Strategy

## Purpose
Framework for scaling location-based landing pages to capture regional search intent.

## Target Architecture
- Base route: `/locations/[city-slug]`

## Page Needs
- Consistent localized H1.
- Specific mentions of the local area.
- Schema markup defining `ServiceArea`.

## Scaling Rules
- Content must be sufficiently localized to avoid duplicate content penalties.
- Include localized map embeds if applicable.
- Route users seamlessly back to the main service funnel.
